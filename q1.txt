Difference between http1.1 and http2

Http1.1:
        > Http1.1 was created in 1997.
        > It does not include any in-built features,so the performance it delivers is less efficient.
        > Http1.1 loads resources one after the other.
        > so if one source cannot be loaded,it blocks all the other resources behind it.
        > It went through several stages of development,this first version of HTTP was called HTTP1.1.




Http2:
      > Http2 was created in 2015.
      > It is designed to be more efficient and performance than http1.1.
      > Http2 is able to use a single TCP connection to send multiple streams of data at once.
      > so one resource blocks any other resource.   
      > In 2015,a new version of HTTP called HTTP2.
      > Http2 solves several problems that the creators of http1.1.
